Assignment #1

Unit Converter


This assignment asks for the user to input measurements to be converted. This calculator converts Celsius to Fahrenheit, Fahrenheit to Celsius, Meters to Feet, and Feet to Meters


Assignment #2

Income Tax Calculator


This assignment asks for the user to input an amount in Philippine peso. The income tax amount will be shown after entering the amount.

Assignment #3

Mathematical Operators


This assignment asks for the user to input a number. The converter shows the Factorial, Sum, and Average of First N Natural Numbers.

Assignment #4

Simple Payroll


This assignment asks the user to input an employee name, number of days worked, daily rate, and deduction amount. A table displaying the number, employee name, number of days worked, daily rate, gross pay, deduction amount, and net pay of the employee.
